# Cosmic-Ray-Simulation

Here's a Cosmic Ray Simulation I coded for a scientific computing class.  The simulation models downscattering caused by a single cosmic proton colliding with an N-14 nucleus in the atmosphere.  The physics is a little bit simplified.  For example, we use a limited verson of the standard model, and we also neglect charge. 

Although the majority of code was developed by myself, the entire mainloop, as well as function outlines were provided by the Spring 2026 CU Boulder PHYS 2600 instructional team.  This team consists of Professor Ivy Tan, Professor Noah Finkelstein, TA Jacob Sitison, and LA Benjamin Stephenson.  Also, any lines of code written with the help of AI is disclosed in a comment near the line.
